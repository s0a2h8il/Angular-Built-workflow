Secrets

Never commit sensitive data like API keys or tokens. Use Repository Secrets.

Using Secrets

  1. Go to your repository on GitHub.
  2. Click Settings > Secrets and variables > Actions.
  3. Click New repository secret.
  4. Add a Name (e.g. MY_API_KEY) and Value.

Accessing in Workflow

steps:
  # Pass as an environment variable
  - name: Run script with secret
    env:
      API_KEY: ${{ secrets.MY_API_KEY }}
    run: node script.js
