Frequently Asked Questions

Is GitHub Actions free? +

Yes! It is completely free for public repositories. For private repositories, you get a certain amount of free minutes per month (e.g., 2,000 minutes for Free accounts) based on your GitHub plan.

Can I run workflows locally? +

Yes, you can use tools like act to run your GitHub Actions locally. It uses Docker to verify your workflows before pushing them to GitHub.

What are the limits? +

There are limits on concurrent jobs (20 for free plan) and job execution time (6 hours max per job).

Can I build Docker images? +

Absolutely. GitHub Actions has excellent support for building and publishing Docker images to Docker Hub or GitHub Container Registry (GHCR).
