Angular CI Example

Here is the complete workflow we setup for this project.

.github/workflows/angular-ci.yml

name: Angular CI

on:
  push:
    branches: ["main"]
  pull_request:
    branches: ["main"]
  workflow_dispatch:

jobs:
  build:
    name: Build Angular App
    runs-on: ubuntu-latest

    steps:
      - name: Checkout repository
        uses: actions/checkout@v4

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: 'lts/*'
          cache: 'npm'

      - name: Install dependencies
        run: npm ci

      - name: Build production
        run: npm run build -- --configuration=production

Key Takeaways

  • npm ci: Always use npm ci instead of npm install in CI/CD for faster, reliable builds.
  • Caching: actions/setup-node has built-in caching for npm.
  • LTS: Using 'lts/*' ensures we are on a stable Node.js version.
